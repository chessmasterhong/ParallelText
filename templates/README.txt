base.html
Acts as placeholder for all other contents via templates. Current placeholders includes:
    - Header
    - Main Navigation links
    - Body content
    - Footer

header.html
Contains logo, theme/slogan and system navigation links. System navigation links includes:
    - Login link
    - Register link
    - User Account link

mainnav.html
Contains main navigation links of the site features the user can have access to.

footer.html
Contains copyright, terms of service, privacy policy, contact information

index.html
Contains home page contents in body of base template.

login.html
Contains login page contents in body of base template.

register.html
Contains register page contents in body of base template.
